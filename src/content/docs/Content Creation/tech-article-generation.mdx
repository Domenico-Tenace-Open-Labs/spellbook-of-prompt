## Description

This prompt is designed to generate a tech article in Markdown format with a title, description, and content.

## Prompt

```markdown

Write an article where I discuss [topic].

Use an informal tone, the [language], and commas instead of dashes (-)
Don't be as wordy as the other articles; try to be a little more concise.
The article will be published on [platform].
```

## Output Example

"Write an article where I discuss Google AI Studio. Use an informal tone, the English, and commas instead of dashes (-)
Don't be as wordy as the other articles; try to be a little more concise.
The article will be published on my personal blog."

```markdown

A markdown article about Google AI Studio.

```

## Explanation

### Key Features

1. **Clear Brief**: The prompt gives the model a direct topic, tone, language, and publication context, making the output more focused.
2. **Concise Style Control**: It explicitly asks for a less verbose result, helping the article stay compact and readable.
3. **Format Guidance**: The instruction to write in Markdown makes the output easier to reuse in blogs, docs, and content systems.
4. **Platform Adaptation**: Because the output is meant for a specific platform, the writing style can be adjusted to fit the audience and publishing context.
5. **Flexible Customization**: You can easily replace placeholders like [topic], [language], and [platform] to generate different articles quickly.

### Best Use Cases

This prompt is ideal for:

- ✅ Bloggers who want fast article drafts
- ✅ Technical writers creating educational content
- ✅ Founders publishing thought leadership posts
- ✅ Marketers producing consistent content for different channels

### Tips to Get the Best Results

- Choose a specific and relevant topic
- Define the target audience clearly
- Pick a language and tone that match the publication
- Review the generated Markdown and adjust structure or examples if needed
