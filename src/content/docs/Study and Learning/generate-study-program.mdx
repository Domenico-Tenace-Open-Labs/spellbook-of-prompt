## Description

This prompt is designed to generate a personalized study program based on your learning goals and available time.

## Prompt

```

I want to learn [skill] from scratch. Create a detailed 30-day plan with daily exercises, essential resources, and milestones to track my progress.

```

## Output Example

I want to learn how to become a developer from scratch. Create a detailed 30-day plan with daily exercises, essential resources, and milestones to track my progress.

```markdown
If you're starting from zero, the fastest way to make progress is to build things every day rather than spending weeks watching tutorials. A month won't make you a professional developer, but it can give you a solid foundation and enough confidence to keep learning independently.

For this plan, I'll assume your goal is **web development**, using **Python** to learn programming fundamentals and **HTML/CSS/JavaScript** for building websites. These technologies are beginner-friendly and widely used.
```

## Explanation

### Key Features

1. Personalized pacing and focus: the prompt tailors topics, exercises and milestones to the learner's goal and available daily time.
2. Day-by-day actionable plan: each day includes concrete exercises, mini-projects or practice tasks to build skill incrementally.
3. Curated resources and milestones: recommends essential learning resources and measurable checkpoints to track progress.
4. Project-based consolidation: includes mini-projects and a capstone to apply knowledge and produce portfolio pieces.
5. Flexible and repeatable: easily re-run or adjust the plan for different durations, difficulty levels, or learning preferences.

### Best Use Cases

- Beginners learning a new skill from scratch
- Learners restarting after a break who need structured momentum
- Short, intensive learning sprints (30-day challenges)
- Educators and coaches designing compact curricula or bootcamp modules

### Tips to Get the Best Results

- Provide the target `skill`, current `level` (e.g., beginner), and daily available `hours`.
- Mention preferred languages, tools, or constraints (e.g., "no paid resources").
- Ask for adjustments (slower pace, more projects, different resources) in follow-up prompts.
- Use the output as a living plan: review weekly, mark milestones, then regenerate the next 30 days with updated goals.

This prompt produces a practical, ready-to-follow 30-day study program that you can adapt iteratively based on progress and feedback.
